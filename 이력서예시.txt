자기소개

저는 블록체인 기술에 ~~한 이유로 관심이 많아 따로 시간이 날 때 마다 학습을 했습니다.
특히 블록체인 기술을 제대로 배우고 싶은 생각에 오픈제플린이나 공식문서와 같은 검증된 소스들을 이용하여
블록체인의 기본적인 이론부터 각종 토큰 표준들, 다양한 종류의 NFT, Openzepplin의 EIP712 서명과 업그레이더블 컨트렉트
Governor와 ERC721 Vote, Multisig와 같은 다양한 컨트렉트를 시간이 날 때 마다 분석 중입니다.

제가 특히 자신 있는 분야는 컨트렉트에 대한 분석을 바탕으로 프론트엔드/ 백엔드 API 제작에 참여하는 것입니다.
블록체인 기술을 배우는데 있어서 항상 진심이고 관심이 많기 때문에 저의 목표는 00회사와 같이 전문적인 블록체인 및 백엔드 개발자 분들과 같이
일할 수 있는 곳에서 개발하는 것입니다.


참여한 프로젝트

- openzepplin의 표준 토큰들 분석 및 정리 (2023년 4/22 ~ 5/22)

    - ERC721, ERC721A, ERC1155를 비롯한 NFT 배포 및 테스트
    - EIP721, Governor를 비롯한 오픈제플린 주요 코드 배포 및 테스트
    - gnosis의 Multisig 지갑 배포 및 테스트
    - 컨트렉트 테스트 코드 작성 (truffle, hardhat 환경에서 모두 작성)
    - 컨트렉트 실행 스크립트 작성 (배포, 함수 실행, 조회 스크립트 작성)
    - 해당 컨트렉트 개념 정리 (readme에 개념 정리)

사용기술 : truffle, web3, ethers, hardhat, javascript, typescipt, solidity

- Dao와 nft를 이용한 크라우드 펀딩 시스템 구현(2023년 6/20~7/10)

    - erc20 토큰의 ICO 페이지 제작 및 컨트렉트 연동
    - nft 구매 페이지 제작 및 컨트렉트 연동
    - governor 컨트렉트를 이용한 DAO 관련 페이지 제작 및 컨트렉트 연동
    - multisig를 이용한 관리자 페이지 작성

사용기술 : metmask, bootstrap, node.js, solidity, javascript, ejs
